SETUP

Upload semua isi folder ini ke lokasi yang sama dengan web panel kamu.

File utama:
- index.html = file kamu + tambahan tab Web Panel / Module Shell
- tweak.json = tetap untuk Web Panel lama kamu
- shell.json = database Module Shell
- download.html = penerima query download ZIP shell
- shell/ = tempat semua ZIP module shell

Jumlah module shell: 220
Module premium: 124
Module free: 96

Fitur:
- Label UI Support Non Root dan Support Root.
- Premium shell random 4 directlink lalu open Chrome setelah sukses.
- Free shell tidak buka iklan dari script.
- Setiap ZIP punya execute.sh, agresif.sh, del.sh, README.txt.
- Script tidak disable package sistem acak, tidak hapus data aplikasi user, tidak edit system/vendor.
