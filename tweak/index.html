Validasi service

neon doctor
neon web-stop && neon web

Local API Contract

Base endpoint

http://127.0.0.1:8787/cgi-bin/api

Gunakan GET untuk action sederhana. Gunakan POST untuk action yang membutuhkan input panjang seperti shell, busybox, toybox, pm, am, dumpsys, settings, dan input.

ParameterWajibDipakai UntukContoh
actYaNama action API yang ingin dijalankan.act=deviceinfo
actionOpsionalAlias dari act.action=apihealth
cmdUntuk shell bridgeIsi command untuk shell, busybox, toybox, run.cmd=pm list packages
pkgUntuk appsPackage name aplikasi.pkg=com.android.settings
nameUntuk moduleID/nama module.name=shell-047
x,y,durationUntuk inputKoordinat tap/swipe dan durasi gesture.x=500&y=1200
textUntuk input teksTeks yang akan dikirim ke input Android.text=halo

Request Examples

curl -i "http://127.0.0.1:8787/cgi-bin/api?act=apihealth"
curl "http://127.0.0.1:8787/cgi-bin/api?act=deviceinfo"
curl -X POST "http://127.0.0.1:8787/cgi-bin/api" -d "act=shell" --data-urlencode "cmd=pm list packages"
curl -X POST "http://127.0.0.1:8787/cgi-bin/api" -d "act=dumpsys" --data-urlencode "cmd=battery"

Response Format

Success

API mengembalikan output plain text agar kompatibel dengan output command Android, BusyBox, Toybox, logcat, dan report.

Error

Jika action tidak dikenali atau parameter kurang, API mengembalikan pesan error plain text dengan nama action yang diproses.

Recommended Client Handling

Client cukup membaca response sebagai text, lalu tampilkan dalam console/log view dengan overflow internal.

Operational Checklist

Start Local API

Gunakan neon web-stop && neon web setelah update paket agar service menggunakan versi terbaru.

Verify Endpoint

Tes dengan curl -i "http://127.0.0.1:8787/cgi-bin/api?act=apihealth".

Use Quoted POST

Untuk command dengan spasi, gunakan --data-urlencode "cmd=..." agar parameter tidak pecah.

Output Location

Media, log, report, dan backup APK disimpan di /sdcard/Download/NeonEngine.
