Installation Guide

1. Siapkan Termux

pkg update -y && pkg install curl unzip -y && termux-setup-storage

2. Download paket ZIP

curl -L https://neonmagisk.my.id/neon-engine.zip -o neon-engine.zip

3. Extract dan jalankan installer

unzip -o neon-engine.zip && chmod +x neon-engine.sh && sh neon-engine.sh

4. Validasi engine dan jalankan API lokal

neon doctor && neon install && neon api

Developer API

Base URL

API berjalan lokal di perangkat user setelah neon api.

http://127.0.0.1:8787/cgi-bin/api

GET Request

curl "http://127.0.0.1:8787/cgi-bin/api?act=deviceinfo"

POST Request

curl -X POST "http://127.0.0.1:8787/cgi-bin/api" -d "act=shell" --data-urlencode "cmd=pm list packages"

JavaScript Client

fetch('http://127.0.0.1:8787/cgi-bin/api', { method: 'POST', headers: {'Content-Type':'application/x-www-form-urlencoded'}, body: new URLSearchParams({act:'shell', cmd:'dumpsys battery'}) }).then(r => r.text()).then(console.log)

Operational Checklist

Engine Check

Jalankan neon doctor. Jika engine belum aktif, aktifkan Neon Core lalu jalankan neon install.

API Check

Jalankan neon api, lalu test act=apihealth dari browser atau curl.

Output Folder

Media, log, APK backup, dan report disimpan di /sdcard/Download/NeonEngine.

Troubleshooting

API 404

Jalankan neon api-stop && neon api. Pastikan URL memakai /cgi-bin/api.

Daemon tidak merespons

Aktifkan Neon Core, lalu ulangi neon install dan neon doctor.

Permission storage

Jalankan ulang termux-setup-storage, lalu izinkan akses file.
