Termux Control + Developer API

Neon Engine Developer Bridge

Neon Engine menyediakan WebGUI lokal dan command neon di Termux agar user, developer aplikasi, atau developer web lokal bisa menjalankan device info, package tools, media tools, log tools, dan shell command melalui API lokal.

Input

WebGUI / Client App → API Request → Termux → Daemon

Output

Daemon Process → Termux → API Response → WebGUI / Client App

Quick install

pkg update -y && pkg install curl unzip -y && termux-setup-storage
curl -L https://neonmagisk.my.id/neon-engine.zip -o neon-engine.zip && unzip -o neon-engine.zip && chmod +x neon-engine.sh && sh neon-engine.sh
neon doctor && neon install && neon web
1

Install ZIP

Download neon-engine.zip, extract, lalu jalankan installer dari folder hasil extract.

2

Validate

Gunakan neon doctor untuk mengecek Termux, tool, WebGUI, dan daemon.

3

Operate

Jalankan neon web, buka 127.0.0.1:8787, lalu pakai WebGUI.

4

Integrate

Client app bisa request ke /cgi-bin/api untuk device info, shell, apps, media, dan logs.

User Installation Guide

1. Siapkan Termux

pkg update -y && pkg install curl unzip -y && termux-setup-storage

2. Download paket ZIP resmi

curl -L https://neonmagisk.my.id/neon-engine.zip -o neon-engine.zip

3. Extract dan jalankan installer

unzip -o neon-engine.zip && chmod +x neon-engine.sh && sh neon-engine.sh

4. Validasi dan jalankan WebGUI

neon doctor && neon install && neon web

Developer API Standard

curl http://127.0.0.1:8787/cgi-bin/api?act=deviceinfo
curl -X POST http://127.0.0.1:8787/cgi-bin/api -d act=shell --data-urlencode cmd="pm list packages"
fetch('http://127.0.0.1:8787/cgi-bin/api', { method: 'POST', headers: {'Content-Type':'application/x-www-form-urlencoded'}, body: new URLSearchParams({act:'shell', cmd:'dumpsys battery'}) })

Core API Actions

Health

apihealth, doctor, features, tools

Shell Bridge

shell, busybox, toybox, pm, am, cmd, dumpsys, settings, input

Device

deviceinfo, battery, storage, memory, cpu, thermal, props

Apps

apps, userapps, sysapps, appinfo, appstart, appstop, backupapk

Media

screenshot, record, tap, swipe, text, key

Logs

logerr, logsave, bugmini, report, openfolder
