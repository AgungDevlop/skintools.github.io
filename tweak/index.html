Termux Control Deployment

Neon Engine ZIP Setup

Panduan ini dibuat untuk instalasi paket neon-engine.zip. Paket ZIP berisi launcher Termux dan Web Control sehingga user cukup extract, jalankan installer, lalu membuka panel lokal melalui browser.

Input

WebGUI → API → Termux → Daemon

Output

Daemon → Termux → API → WebGUI

Port

127.0.0.1:8787

Install command

pkg update -y && pkg install curl unzip -y && termux-setup-storage
curl -L https://neonmagisk.my.id/neon-engine.zip -o neon-engine.zip && unzip -o neon-engine.zip && chmod +x neon-engine.sh && sh neon-engine.sh
neon doctor && neon install && neon web
1

Prepare

Update package, install curl dan unzip, lalu aktifkan akses storage Termux.

2

Deploy ZIP

Download neon-engine.zip, extract, lalu jalankan neon-engine.sh dari folder hasil extract.

3

Validate

Gunakan neon doctor untuk memastikan Termux command, Web Control, dan daemon siap.

4

Operate

Jalankan neon web, buka alamat lokal, lalu kontrol fitur dari WebGUI.

Enterprise Deployment Guide

1. Siapkan environment Termux

pkg update -y && pkg install curl unzip -y && termux-setup-storage

2. Download paket resmi dalam format ZIP

curl -L https://neonmagisk.my.id/neon-engine.zip -o neon-engine.zip

3. Extract paket dan jalankan installer

unzip -o neon-engine.zip && chmod +x neon-engine.sh && sh neon-engine.sh

4. Validasi koneksi control channel

neon doctor

5. Pasang atau repair komponen Neon Engine pada daemon

neon install

6. Jalankan Web Control

neon web

Operational Contract

Input Request

User menekan tombol di WebGUI. Panel mengirim request ke API lokal dengan parameter action dan input yang dibutuhkan.

Termux Control

Termux menerima request, menjalankan command neon, lalu meneruskan proses ke daemon Neon Core yang sedang aktif.

Response Delivery

Output daemon dikembalikan ke Termux, diteruskan sebagai API response, lalu ditampilkan di WebGUI sebagai live response.

WebGUI Capability

API Health

Cek channel WebGUI, API lokal, Termux, dan status daemon.

Module Manager

Refresh, search, info, download, execute, agresif, dan delete module.

Device Control

Device info, battery, storage, memory, CPU, thermal, props, uptime, dan service list.

Apps Control

List package, app info, start, force stop, appops, dan backup APK.

Media Input

Screenshot, screen record, tap, swipe, input teks, keyevent, clipboard, dan notification.

Logs Report

Logcat, error log, bug mini, full report, dan akses folder output.

Production Checklist

Package

neon-engine.zip sudah ter-extract dan file neon-engine.sh serta webgui.html berada di folder yang sama.

Termux

curl, unzip, dan storage permission tersedia.

Daemon

Neon Core aktif sebelum menjalankan neon install atau fitur yang membutuhkan proses daemon.

Web Control

Panel dibuka dari http://127.0.0.1:8787 setelah command neon web aktif.

Troubleshooting

API Offline

Jalankan neon web-stop && neon web, lalu buka ulang alamat lokal.

Daemon Belum Aktif

Aktifkan Neon Core terlebih dahulu, kemudian jalankan neon install dan cek ulang dengan neon doctor.

Module Gagal Download

Pastikan koneksi internet aktif, lalu jalankan neon refresh sebelum download ulang module.

Output Tidak Terlihat

Cek folder /sdcard/Download/NeonEngine untuk media, APK, log, dan report.
